Project 1: Binary Translator

The first project I practiced python on was this Binary Translator code that translates text to binary and binary to text. The code features the support of regular text and numbers and the translated text is given an indent to provide separation and clarity upon translation.

Project 2: Dice Code

This project features a simple dice program where the user can control how many number of die are present or being used, the user can then roll with the amount of die present and then the code will print out the number rolled by all the die.

Project 3: Sodoku Solver

This is one of the more complex projects that I made and up til now the way on how I made it still makes me scratch my head. This program features a sodoku solver where the user can place the sodoku puzzle onto the code and the software will automatically solve the puzzle.
